
/etc/passwd : print all users

/etc/group : print all groups

etc : editing text config (files de config) de tous les programmes & commandes 

opt : données & programmes optionnels

proc : processus en cours

run : infos systemes depuis dernier demarrage du os 

srv : service tournant en tache de fonds

usr : binaire d'usage (binary des programmes) courant

tmp : files temporaires

bin  : binaires d'execution des programmes & commandes

bin  : super binaires d'execution necessaires au fonctionnement de os

var : données des programmes

boot : contient les files du chargeur d'amorcage | gere le dual boot

dev : devices liés au os

lib : regroupe tous les programmes necessaires au fonctionnement du OS

home : contient les repertoires personnels de chaque user du OS [Bureau, Downloads, Images....]

media : contient tous les medias [clés, DD ] connectés au OS  -
         Point de montage de medias amovibles se fait dans media
